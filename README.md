# Читалка с переводом

Flutter-приложение для чтения электронных книг со встроенным переводом.

## Текущий статус
- Этап 1 завершён: добавлены правила проекта и каркас интерфейса читалки.
- Этап 2 в работе: реализованы выбор файла (`txt/fb2/epub`), постраничный просмотр, возврат к книге в рамках текущей сессии и перевод слов/фраз в режиме чтения.
- Следующий шаг: подключение перевода выделенного текста.

## Правила проекта
- Подробные правила находятся в [`PROJECT_RULES.md`](PROJECT_RULES.md).
